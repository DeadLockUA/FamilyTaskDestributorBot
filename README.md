# FamilyTaskDestributorBot
This is a telegram bot to share and destribute tasks in the family. Perform some planning and track progress.

# How to run it
You need to install python-telegram-bot.

pip install python-telegram-bot

If you use Python 3.12 and pip doesn’t work:

python -m pip install python-telegram-bot

Or:

py -m pip install python-telegram-bot
